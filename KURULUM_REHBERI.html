GitHub'a ücretsiz hesap aç
Ücretsiz · 2 dk
🌐
github.com'a git ve kayıt ol
E-posta ve şifre ile ücretsiz hesap aç. GitHub sonsuza kadar ücretsiz.
github.com/signup →
➕
Yeni repository oluştur
Sağ üstteki "+" → "New repository" tıkla. İsim: biz-birlikte · Public seç · README ekle tıkla → "Create repository"
📤
Dosyaları yükle
"Add file" → "Upload files" tıkla. ZIP'ten çıkardığın TÜM dosyaları (index.html, manifest.json, sw.js, icons klasörü) sürükle. "Commit changes" tıkla.
2
Netlify ile internete yayınla
Ücretsiz · 3 dk
🔗
app.netlify.com/start'a git
"GitHub ile Giriş Yap" seç. GitHub hesabını bağla.
app.netlify.com/start →
📁
Repository seç
"biz-birlikte" repo'sunu seç. Deploy settings'de hiçbir şeyi değiştirme → "Deploy site" tıkla.
✅
Linkini kopyala!
Netlify sana şöyle bir link verir:
https://amazing-couple-123abc.netlify.app
💡 Bu linki ikiniz de telefonunuzda açabilirsiniz! Hemen çalışır.
3
PWABuilder ile APK oluştur
Ücretsiz · 5 dk
🏗️
pwabuilder.com'a git
Microsoft'un ücretsiz aracı. Netlify linkini giriş kutusuna yapıştır → "Start" tıkla.
pwabuilder.com →
📊
Puanı kontrol et
PWABuilder uygulamanı analiz eder. Manifest ve Service Worker hazır olduğu için yüksek puan alacaksın (100+).
📱
Android paketi oluştur
"Package for stores" → "Android" seç → "Generate Package" tıkla. Birkaç dakika bekle → .apk ve .aab dosyalarını indir.
📲
APK'yı direkt yükle (Google Play olmadan)
APK dosyasını WhatsApp ile partnerine gönder. Telefonda aç → "Bilinmeyen kaynaklardan yükle" izni ver → Kur! İki telefonunuzda da çalışır.
💡 Ayarlar → Güvenlik → Bilinmeyen Kaynaklar → Aç
